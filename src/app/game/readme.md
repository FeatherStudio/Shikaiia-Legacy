# Noun
名词约定

## Dungeon Master
即*用户*操作的**玩家**

## Character
即*玩家*操作的**角色**

## Equipment
即*玩家*收集的、*角色*穿戴的**装备**，可以提供*角色*的*战斗牌*

## Equipment Deck
即*角色*可使用的**战斗牌**

## Extra Deck
即*角色*可使用的**额外牌**

## Skill Deck
即*角色*可使用的**技能牌**

## Master Deck
即*玩家*可使用的**城主牌**

## Master Skill Deck
即*玩家*可使用的**城主技能牌**

## Battlefield Deck (optional)
即*战场*可使用的**区域牌**
